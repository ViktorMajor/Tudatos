# Tudatos szülők

Ez a projekt egy statikus weboldal szülők számára. Célja a pszichoedukáció támogatása barátságos, áttekinthető felületen keresztül.

A weboldal tartalmaz bemutatkozást, az alkalmak menetét, videós kísérleteket, letölthető anyagokat, valamint egy egyszerű kvízt is.

Futtatáshoz csak nyisd meg az `index.html` fájlt egy böngészőben.
